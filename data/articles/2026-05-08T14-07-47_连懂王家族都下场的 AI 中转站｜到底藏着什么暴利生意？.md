# 连懂王家族都下场的 AI 中转站｜到底藏着什么暴利生意？

> **来源**: 微信公众号 · [原文链接](https://mp.weixin.qq.com/s/MgWNAwBDbPNUrBSZc7ZCiQ)
> **作者**: Xuan酱 · **时间**: 2026年5月8日 15:07
> **改写方式**: 规则引擎 (去AI味 + 口语化 + 提高原创度)

---

就最近，懂王家族，推出了一个 AI 中转站平台 WorldClaw。

300 多种模型，官网价打 7 折，买顶配套餐还能抽奖去海湖庄园，参加懂王的私人晚宴。

五一假期前后，国内也有两位大佬官宣入局——

币圈传奇孙宇晨，下场做了 B.ai。

猎豹移动董事长兼 CEO 傅盛，推出 EasyRouter。

	

你不好奇吗？从某鱼上卖 Key 的小贩，到孙宇晨、懂王家族，大家排着队往这个赛道里冲。

这个叫「AI 中转站」的东西，到底是什么？里面又藏着什么生意？

中转站到底是什么

如果你不写代码、不调 API，没用过 Claude Code、OpenClaw 那些，你可能就没听说过中转站。

它的本质，说白了就是「倒爷」。

帮国内用户代购海外 AI 服务，再通过信息差、汇率差、渠道折扣和灰色玩法等等赚钱。

中间商赚差价，古老的生意逻辑，套了一层 AI 的皮。

具体怎么个倒法？

对于国内的独立开发者或者中小型 AI 创业团队来说，想直接调海外大模型的 API，要过好几道坎：直连不稳定、国内信用卡付不了款、账号三天两头被封……

中转站就是专门帮这批人解决这些破事的，链路变成了：

「用户 ↔ 代购（中转站）↔ 官方」。

它帮你解决了支付难、网络差、易封号的问题。

代价是——你所有的 API 请求，都得先在中转站的服务器上过一遍。

听着就不太安心——但这门生意，偏偏火得一塌糊涂。

为什么这个生意会这么火

需求太大了

先看一组数据，保证你和我一样看完眼珠子都要掉出来。

OpenRouter 是目前全球最大的 AI 中转站，团队不到十个人，a16z 却给出了 5 亿美元的估值，去年直接领投 4000 万美金。人家年流水早已过亿。

十个人不到，做到这个规模——难怪谁都想来分一杯羹。

以他们最近一周的 LLM 使用榜单来看，光是榜单前 10 个模型，一周合计就干掉了约 12.6 万亿 tokens，折合日均 1.8 万亿。

这串数字看着抽象，换算成钱就直观了。

以 Claude Sonnet 4.6 为例，官方定价约 3 美元/百万 tokens。它本周在 OpenRouter 上的用量是 1.36 万亿 tokens。

粗算下来，一周约 400 万美元，折合人民币将近 3000 万元。

这还只是单个模型、单平台、单周的数字。

去掉免费模型、按付费模型均价保守估算，仅 OpenRouter 一个平台，每周流水级别就在数千万美元量级。

这个数字意味着，AI 调用早已不是极客圈的小众需求，它正在变成一种像水电一样的日常消耗。

每周有数以万计的开发者，在花真金白银调 API。

这还只是一个平台的公开数据。整个中转市场的体量，只会更大。

利润空间更大

愿意花钱的人多，想从中间捞钱的人自然更多。

AI 中转站的利润结构，恰好又对操盘手极其友好——它不是苦哈哈的薄利多销，而是各种套利手段的叠加。

其中相对「正经」，处在灰色边缘的，是资源端的套利，和技术端的套利。

先说资源端。就是利用地区低价、初创补贴、汇率波动等等各种方式，用更低成本拿到 AI 额度，再按市场价卖出去。

最直接的一层，是地区定价差。

AI 服务的全球定价，是严重不均衡的。同一个 ChatGPT Plus，美国用户掏 20 美元一个月，阿根廷、土耳其、尼日利亚这些低收入地区的用户，可能只要 4 到 8 美元。

差价就摆在那里，谁有办法用低价区的价格拿到货，再按高价区的市场价卖出去，谁就能吃到这层利润。

地区差价之外，还有一条更暴利的路子——白嫖补贴。

有些云厂商会给初创公司送 AI 服务的额度。比如 Google Cloud 为了拉拢 AI 初创公司，最高送出 35 万美元的免费额度，两年内随便用。

这个漏洞，当然有人会钻。有资源的人就会注册一堆壳公司，专门去套这笔补贴。拿到免费额度之后，转手卖给中转站当上游货源。

成本是零，卖出去的全是纯赚。羊毛薅到飞起。

再说技术端。这一层更隐蔽，不靠地域差价，而是靠工程手段把成本压下去。

逻辑很简单：像OpenAI、Anthropic他们其实提供了很多折扣选项，但都附带各种条件——有点像网盘的那种闲时下载包。

普通用户懒得研究、也用不好。中转站就把这些折扣全部吃透，压低自己的真实成本，对外却照样收全价。

具体有几种常见玩法——

一、Batch API（批量处理）

比如 OpenAI 的 Batch API ，官方直接给 50% 折扣，成本直接砍半。

但相应的，请求变成异步处理，最长需要等 24 小时才能拿到结果。

中转站只需要在后端做好调度，把不紧急的请求悄悄走 Batch 通道。

用户感觉速度慢，钱却照全价交，实打实地进了中转站口袋。

二、Prompt Cache（提示词缓存）

如果你每次发给模型的请求，开头都有一大段重复内容（比如设定角色的指令），模型其实不需要每次都重新"读"一遍。

开启缓存之后，这部分 token 的费用会大幅低于普通输入价格。比如 OpenAI 的，直接就是普通输入的一折。

中转站在工程层面做好缓存策略，实际成本能压低不少，但用户那边仍然按普通价格计费。

上面这些，都还只是冰山一角，都还算是「正经生意人」的操作。

更下面的，就真的是纯纯的黑心了。

比如盗刷信用卡。有人手里攥着大批黑卡号，利用 OpenAI、Anthropic 等平台在支付验证环节的漏洞，用脚本批量注册账号。号被封了就再开，除了电费和脚本跑的时间，几乎没成本。

中转站黑幕

盗刷信用卡，都还只是管中窥豹。

中转站的黑幕，真的，无穷无尽。。

下面我要讲的，是目前能找到可靠资料支撑的；有些暂时还没有直接证据，但已经是业内公认的黑幕了。

模型掉包：你花顶级模型的钱，跑的是免费小模型

去年 3 月，CISPA 亥姆霍兹信息安全中心发了一篇调查报告，上来就给整个行业扎了一针。

他们审计了 17 家市面上活跃的中转站，将近 50% 在模型身份上做了手脚。

换句话说，你花钱买的是 Claude Opus，路由脚本判断你任务复杂之后……直接把你的请求扔给一个免费的国产开源小模型。

这还不算完。还有两招更阴的——

一招是偷偷砍上下文窗口。Claude 理论上的上下文窗口最高能到 200K tokens，到了某些站手上，后台直接给你截到 32K。

另一招是计费倍率做手脚。官方实际消耗 100 个 token，后台报表给你写成 300 个。

单看每一项，都是小动作。但把这几招叠在一起——用便宜模型顶替、把上下文窗口掐窄、再把计费倍率一放大——你来算算他们的毛利是多少？

倒卖数据：你的每一次提问，都是别人拿去卖钱的语料

中转站另一个让人后背发凉的地方，就是你的所有请求和返回，在中转站服务器那里全是明文。

X上有人爆料，有用户通过中转站调用 ChatGPT，对话记录不知道怎么就被泄露了出去。

最终他花了 3 万块钱，才把这件事压下去。

来源：X @ai_xiaomu 《API中转站,比贩毒还赚》

但这还只是个人倒霉的遭遇。更系统性的玩法，可能是这样的——

所有经过中转的请求，完整的输入输出，对 AI 厂商来说，就是一份现成的蒸馏数据。

尤其是 Claude Code 这类重度编程场景，用户产出的不是闲聊废话，全是复杂的推理链条、真实的工程决策、多轮修 bug 的过程记录。这种数据，是模型厂商做梦都想要的训练素材。

所以，中转站真正的商业模式，很可能是这样的：

收你的中转费，只是表面生意。把你源源不断产生的请求数据，打包卖给大厂做模型蒸馏，才是核心利润来源。

你是付费客户。同时，也是免费的训练数据生产者。一鱼两吃。

再往深了说一句——有些中转站上线，核心目的根本就不是卖 API。卖 API 挣几个钱？

它们的定价之所以可以那么低，是因为低价本身就是获客手段。

用户涌进来产生的海量高质量对话，才是它真正要拿去变现的产品。

来源：X @yan5xu

预充值跑路：有些站从第一天就没打算长期运营

除了数据被偷摸着倒卖，还有一种损失更直接——钱充进去了，站没了。

今年一月就有个案例，Privnode。这个中转站崩盘之前，上演了一出荒诞大戏。

它把整座站以 10 块钱人民币的价格，卖给了一个所谓的接盘人。

而那个时候，用户账户里还有 66 万余额没花完，170 多笔退款没处理。

接手后，接盘人立刻失联。

更绝的是——接盘人还是个初中生。

就这种情况，一般钱都要不回来了。跟健身房跑路真的是异曲同工。。

注入恶意代码：你以为它在帮你写代码，其实它在搞你的机器

上面说的种种，要么是骗钱，要么是偷数据。但接下来这一层，直接指向了你机器的控制权。

加州大学的安全研究团队发布了一篇论文《Your Agent Is Mine》。

他们对市面上 28 个付费 API 中转服务和 400 个免费路由服务做了系统性测试。这些服务的来源覆盖了淘宝、闲鱼、Shopify 店铺和大量公开社区。

结果触目惊心。

有 1 个付费中转站和 8 个免费中转站，会主动在返回内容里注入恶意代码。你闭着眼敲回车，一个带着勒索病毒或者挖矿程序的恶意依赖包，就会被悄悄安进你的系统根目录。

有 17 个中转站，触碰了研究者故意布置的 AWS 诱饵凭证。通俗来讲，就像你托人送一封信，送信的人不仅拆开看了，还复制了信里夹着的一把钥匙，跑到你家门口去试了试锁。

还有 1 个中转站，在检测到研究者放置的 ETH 钱包私钥之后，直接把资金一键清空，全部转走了。

你的电脑、你的密钥、你的钱包，在这些人眼里，都是猎物。

写在最后

这行里有太多人，从一开始就没打算好好做生意。

他们建站的第一天，想的不是怎么服务用户，而是怎么从用户身上榨干最后一滴价值。

再加上这个市场几乎没有监管，没有准入门槛，没有惩罚机制。认真做的和忽悠人的，在用户眼里几乎长得一模一样。

所以我们更加需要知道风险在哪里，才能真正用得安心。

最后再给大家一些掏心窝子的建议。

能直连官方就直连，能用大平台就优先用大平台。 如果当下阶段非得用中转，优先选 OpenRouter 这类有正规机构背书、有清晰隐私政策的大平台。

用多少充多少。 永远不要一次性往中转站充一大笔钱。

敏感数据绝不走中转。 商业核心代码、用户隐私信息、金融凭证、私钥、未公开的商业计划——这些东西，打死也不要经过任何非官方节点。

第四，所有代码和终端命令，确认安全再跑。养成一个肌肉记忆：AI 输出的重要命令，在自己亲眼确认之前，别敲回车。

如果你手里已经有中转站的 Key，不确定靠不靠谱，可以把临时测试密钥拿去检测工具跑一跑。

比如 https://llmtest.cn/ 和 https://api-check.com/，可以帮你从模型身份、响应一致性等维度做一轮黑盒筛查，看看模型是不是被掉了包。

下单之前想先摸底，也可以去 https://token1000.com/ 这种中转站收录平台，看看真实用户的反馈和口碑评价，尽量绕开那些高风险预警的站点。
